Правила:

  1. Игроки по очереди ставят на свободные клетки поля 4×4 знаки. Один игрок ходит крестиками, а другой – ноликами. Первый, выстроивший в ряд 4 своих фигуры по вертикали, горизонтали или диагонали, выигрывает.
  2. У игроков есть по одной мине, вы можете использовать мину только один раз за игру. Чтобы использовать мину, нужно нажать на кнопку "Мина", и после нужно выбрать один из символов противника в клетке. После нажатия на него, он уберётся с поля игры и ход перейдёт на ход противника
